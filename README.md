# Insurance-Claim-Analysis-Data-Analytics-
The Insurance Claim Analysis is a user-friendly tool designed to explore how demographic factors and health indicators impact insurance claims. Using an interactive platform, it provides insights into claim patterns, helping users understand key trends and relationships in the data
The dashboard covers several aspects, including age and gender distributions, BMI and blood pressure profiles, regional claim trends, smoking status, and family size. It highlights how claim amounts vary by age group and gender, examines the influence of health indicators like BMI and blood pressure, and uncovers regional differences in claims. It also explores the impact of smoking habits and family size on claim amounts.Visualizations include line charts to show how claim amounts change with age, pie charts for regional and behavioral distributions, and bar charts comparing claims across demographics, health categories, and family sizes. A composite bar chart provides a combined view of claim amounts, gender, smoking, and diabetic status.The dashboard is built using Power BI and Python libraries like Matplotlib, Seaborn, and Pandas, ensuring efficient data analysis and clear visual representations.This tool simplifies complex data, making it easier for users to identify patterns and trends in insurance claims. It serves as a valuable resource for decision-making in the insurance industry, helping stakeholders understand factors that influence claim amounts and improve their strategies
